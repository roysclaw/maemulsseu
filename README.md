# 매물쓰 - 부동산 매물 설명문 자동생성 SaaS

Vue 3 + Vite + Tailwind CSS로 제작된 랜딩페이지입니다.

## 기술 스택

- Vue 3 (Composition API)
- Vite
- Tailwind CSS
- Noto Sans KR (Google Fonts)

## 실행 방법

```bash
npm install
npm run dev
```

개발 서버: http://localhost:5173/

## 빌드

```bash
npm run build
```

## 페이지 구성

1. **헤더** - 고정, 로고 + 무료 시작 버튼
2. **히어로** - 메인 헤드라인 + CTA
3. **문제 제기** - 3개 카드
4. **기능 소개** - 3단계 플로우
5. **미리보기** - 입력/출력 예시 (탭)
6. **요금제** - 무료/베이직/프로
7. **사전신청** - 이메일 수집
8. **푸터** - 저작권 + 링크
